# Refresh log

Quarterly refresh cycle. Blog-post refresh is the highest-ROI SEO activity.
Target: pages ranked #4–#20 in GSC, published >90 days ago, no update in >60 days.

## Refresh scoring

Rank by GSC-reported impressions × (position gap from top 3). High-impression

- close-to-top-3 = fastest wins.

## Refresh checklist per article

- [ ] Re-Google the target keyword. Compare current top-3 to our article.
- [ ] Update: statistics, screenshots, product references, dates in copy
- [ ] Add: 2–3 new internal links (from newer articles)
- [ ] Add: any missing sections the top-3 covers that we don't
- [ ] Verify: meta title ≤58, description ≤155, H1 ≤60
- [ ] Verify: cover image alt + filename
- [ ] Update: `updatedAt` field (should happen automatically)
- [ ] Resubmit URL in Google Search Console
